# Mazen Bassiso — Portfolio (Adaptify-inspired)

Static, fast, and GitHub Pages friendly.

## Quick start (local)

Just open `index.html` in your browser.

## Deploy to GitHub Pages

1. Create a new repo on GitHub (example: `mazen-portfolio`)
2. Upload all files in this folder (or push via git).
3. In GitHub: **Settings → Pages**
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
5. Save — GitHub will give you a public URL.

## Customize

- Edit text in `index.html`
- Update styles in `styles.css`
- Replace the monogram avatar with an actual photo:
  - Add an image in `assets/portrait.jpg`
  - Swap the avatar markup with `<img class="avatar__img" src="assets/portrait.jpg" alt="Mazen Bassiso" />`

## Data-driven updates (no code changes)

Edit `data.json` to update:
- Websites list
- Media kit numbers, packages, and add-ons
- Meeting links
- Brands grid items

Then push changes to GitHub — the site updates automatically.
